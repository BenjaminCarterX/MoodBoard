# MoodBoard

一个简洁的情绪追踪工具，帮助记录和可视化每日心情状态。

## 功能特性

- 📊 1-10分情绪评分滑块
- 🏷️ 六种常用情绪标签
- 📝 每日心情备注功能
- 📱 响应式设计，支持移动端
- 💾 本地存储数据
- 📈 历史记录查看

## 使用方法

1. 在浏览器中打开 `index.html`
2. 调整情绪滑块选择今日心情评分
3. 点击选择合适的情绪标签
4. 可选择性添加今日心情备注
5. 点击保存按钮记录今日心情
6. 在历史记录区域查看过往心情变化

## 技术栈

- HTML5
- CSS3 (使用了渐变背景和现代化样式)
- 原生 JavaScript (ES6+)
- LocalStorage 本地数据存储
